# 组件介绍
---

所有主题均配备的UI组件：

- Col           列
- Row           行
- Link          链接
- Label         标签
- Image         图片
- Table         表格
- DataTable     数据表格
- Form          表单
- Tabs          Tab标签显示框
- Popup         弹框 
- Box           显示Box
- Paginator     分页
- Button        按钮
- Tree          树
- Alert         alert弹窗

Adminlte内置了一些ui组件供使用：

- [infobox](components/infobox.md)
- [progress-group](components/progressbar.md)
- productlist
- smallbox
- description
- chart legend

系统提供的额外UI组件：

- [chartjs](components/chartjs.md)
